Create a modern Arabic web application called "عروض أسعار" (Quotation Generator).

The application should be simple, professional, and fast. Its purpose is to take unorganized text from the user and automatically convert it into a structured quotation table.

Main Idea

The user pastes product lines or items as text, and the system automatically organizes them into a table with the following columns:

| # | الصنف | الوصف | الكمية | السعر | الإجمالي |

The system must automatically:

Number the rows.

Calculate the total price (الكمية × السعر).

Calculate the final total for the quotation.

Example Input

If the user writes something like:

مجنونة 3م 30 5
جولد ستار 1.5م 40 3
اكاسيا جيلوقا 3م 15 4

The system automatically converts it to a table.

Features Required

Smart Text Parser

Reads pasted text.

Detects name, description, quantity, and price automatically.

Generates rows in the quotation table.

Editable Table
User can modify:

quantity

price

description

Automatic Calculations

Row total = quantity × price

Grand total at bottom

Export Buttons
Add 3 export buttons:

Export to Excel (.xlsx)
Export to Word (.docx)
Export to PDF (.pdf)

Files should be generated instantly with the formatted quotation table.
